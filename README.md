# Tabuada Usando For Int

Código onde o usuario insere o valor que deseja multiplicar, até quando quiser que essa multiplicação ocorra.

# Exemplo:


Ate quanto deseja multiplicar?  **13**

7 x 0 = 0

7 x 1 = 7

7 x 2 = 14

7 x 3 = 21

7 x 4 = 28

7 x 5 = 35

7 x 6 = 42

7 x 7 = 49

7 x 8 = 56

7 x 9 = 63

7 x 10 = 70

7 x 11 = 77

7 x 12 = 84

7 x 13 = 91
